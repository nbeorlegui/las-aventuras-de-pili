# Las Aventuras de Pili - Next.js

Versión optimizada con imágenes WebP responsive incorporadas.

## Ejecutar local

```cmd
npm install
npm run dev
```

Abrir: http://localhost:3000

## Build

```cmd
npm run build
```

## Netlify

Configuración:

- Build command: `npm run build`
- Publish directory: `.next`

Incluye `netlify.toml`.
